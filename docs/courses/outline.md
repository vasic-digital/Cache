# Course: Caching in Go

Learn to build high-performance caching layers using `digital.vasic.cache` -- from in-memory to distributed multi-level architectures.

## Lessons

1. **Core Interface and In-Memory Cache** -- The `Cache` interface, `TypedCache[T]`, eviction policies, memory limits, and background cleanup.
2. **Redis and Distributed Patterns** -- Redis single/cluster backends, consistent hashing, two-level caching, and write strategies.
3. **TTL Policies and Service Caching** -- Fixed, sliding, and adaptive TTL policies, eviction deciders, and the service-layer cache-aside wrapper.
